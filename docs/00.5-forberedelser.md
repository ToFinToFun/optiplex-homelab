# Förberedelser — Allt du kan göra INNAN din OptiPlex kommer

Den här guiden listar allt du kan förbereda redan nu, medan du väntar på att hårdvaran ska levereras. Genom att göra detta i förväg sparar du timmar på installationsdagen och undviker att sitta fast mitt i processen för att du saknar ett konto eller en nyckel.

## Översikt

| Uppgift | Tid | Kräver | Resultat |
|---------|-----|--------|----------|
| Skapa Cloudflare-konto + flytta domän | 15 min + väntetid | Domännamn | Domän redo i Cloudflare |
| Skapa Cloudflare Tunnel-token | 10 min | Cloudflare-konto | Token sparad |
| Ladda ner Proxmox ISO + skapa USB | 10 min | USB-sticka (≥2 GB) | Bootbar USB |
| Beställ RAM (om ej gjort) | 5 min | — | 16 GB DDR4-sticka på väg |
| Beställ lagringsdisk (om ej gjort) | 5 min | — | SSD på väg |
| Bestäm IP-plan | 5 min | Tillgång till routern | IP-adresser reserverade |
| Förbered kameror (lösenord, nätverk) | 15 min/kamera | Kameror installerade | Kameror redo för Frigate |
| Skapa GitHub-konto (valfritt) | 5 min | — | Kan klona detta repo |
| Fyll i TOKENS.md | 10 min | Alla ovan | Alla nycklar på ett ställe |

## 1. Domän & Cloudflare-konto

**Vad:** Du behöver en domän (t.ex. `dittnamn.se`) som administreras via Cloudflare. All extern åtkomst till din server går via Cloudflare Tunnel.

**Gör nu:**
1. Om du inte har en domän: köp en via [Cloudflare Registrar](https://dash.cloudflare.com/registrar) (billigast, ingen markup) eller valfri registrar.
2. Skapa ett gratis konto på [dash.cloudflare.com](https://dash.cloudflare.com/sign-up) om du inte redan har ett.
3. Lägg till din domän i Cloudflare (se [guide 03.5](03.5-doman-cloudflare.md) för detaljer).
4. **Vänta tills domänen visar "Active"** i Cloudflare-dashboarden. Detta kan ta 10 minuter till 24 timmar beroende på din registrar.

> **Tips:** Gör detta FÖRST. Nameserver-bytet kan ta tid, och du vill inte sitta och vänta på installationsdagen.

**Resultat att spara:**
- Cloudflare-kontots e-postadress
- Domännamn (t.ex. `mitthem.se`)

## 2. Skapa Cloudflare Tunnel (token)

**Vad:** En tunnel är den krypterade anslutningen mellan din framtida server och Cloudflare. Du kan skapa tunneln i förväg — den väntar tålmodigt tills din server ansluter.

**Gör nu:**
1. Logga in på [Cloudflare Zero Trust](https://one.dash.cloudflare.com/).
2. Gå till **Networks** → **Tunnels** → **Create a tunnel**.
3. Välj **Cloudflared** som typ.
4. Döp tunneln till något beskrivande (t.ex. `hem-proxmox`).
5. På nästa sida visas installationskommandon. Du behöver bara **kopiera token-strängen** (den långa texten som börjar med `ey...`). Spara den i din `TOKENS.md`.
6. Du behöver INTE konfigurera "Public Hostnames" ännu — det gör vi när servern är igång.

**Resultat att spara:**
- Tunnel-token (lång sträng som börjar med `ey...`)
- Tunnel-namn

## 3. Konfigurera Cloudflare Access (valfritt men rekommenderat)

**Vad:** Cloudflare Access skyddar dina tjänster med en e-postverifiering (OTP) så att ingen obehörig kan komma åt dem.

**Gör nu:**
1. I [Cloudflare Zero Trust](https://one.dash.cloudflare.com/), gå till **Access** → **Applications** → **Add an application**.
2. Välj **Self-hosted**.
3. Skapa en applikation för Frigate:
   - **Application name:** `Frigate`
   - **Session Duration:** 24 hours
   - **Application domain:** `frigate.dindomän.se`
4. Skapa en policy:
   - **Policy name:** `Tillåt mig`
   - **Action:** Allow
   - **Include:** Emails → din e-postadress
5. Upprepa för `npm.dindomän.se` (NPM-gränssnittet).
6. **Skapa INTE en Access-policy för `ha.dindomän.se`** — Home Assistant-mobilappen hanterar sin egen autentisering och fungerar inte bakom Cloudflare Access.

**Resultat att spara:**
- Vilka subdomäner du valt (t.ex. `ha.dindomän.se`, `frigate.dindomän.se`, `npm.dindomän.se`)

## 4. Ladda ner Proxmox ISO och skapa bootbar USB

**Vad:** Proxmox VE installeras från en USB-sticka, precis som Windows eller Ubuntu.

**Gör nu:**
1. Gå till [Proxmox VE Downloads](https://www.proxmox.com/en/downloads/proxmox-virtual-environment/iso).
2. Ladda ner den senaste **Proxmox VE ISO Installer** (ca 1.2 GB).
3. Ladda ner [balenaEtcher](https://etcher.balena.io/) (gratis, finns för Windows/Mac/Linux).
4. Sätt i en USB-sticka (minst 2 GB, allt innehåll raderas).
5. Öppna Etcher, välj Proxmox ISO-filen, välj USB-stickan, klicka "Flash!".
6. När det är klart, märk USB-stickan "Proxmox" och lägg undan den.

> **Viktigt:** USB-stickan behöver bara användas en gång (vid installationen). Efteråt kan du formatera om den.

## 5. Beställ rätt RAM och lagringsdisk

**RAM:**
- Du behöver **en DDR4 SODIMM-sticka på 16 GB** (OptiPlex XE4 Micro använder laptop-RAM).
- Kontrollera att den matchar hastigheten på den befintliga stickan (troligen DDR4-3200 / PC4-25600).
- Med 2× 16 GB får du **Dual Channel** vilket ger dubbel minnesbandbredd — viktigt för videobearbetning.

**Lagringsdisk (för Frigate-inspelningar):**
- Rekommendation: **1 TB SATA SSD** eller **1 TB 2.5" SSD** (beroende på vad OptiPlexen har plats för).
- OptiPlex XE4 har en M.2-plats (för OS) + en 2.5" SATA-plats (perfekt för lagringsdisken).
- Välj en disk med högt TBW-värde (t.ex. Samsung 870 EVO, WD Blue SA510, eller Crucial MX500).

| Komponent | Rekommendation | Ungefärligt pris |
|-----------|---------------|-----------------|
| RAM 16 GB DDR4-3200 SODIMM | Kingston, Crucial, eller Samsung | ~400-600 kr |
| SSD 1 TB 2.5" SATA | Samsung 870 EVO / WD Blue / Crucial MX500 | ~700-1000 kr |

## 6. Bestäm IP-plan

**Vad:** Varje tjänst på servern behöver en egen IP-adress i ditt hemnätverk. Det är enklast att bestämma dessa i förväg.

**Gör nu:**
1. Logga in i din Unifi-router (eller annan router).
2. Kolla vilket IP-intervall ditt nätverk använder (vanligtvis `192.168.1.x` eller `192.168.0.x`).
3. Välj fyra lediga IP-adresser i den övre delen av intervallet (t.ex. `.100`–`.103`). Dessa ska inte användas av andra enheter.

Fyll i tabellen nedan med dina valda adresser:

| Tjänst | IP-adress | Kommentar |
|--------|-----------|-----------|
| Proxmox (värden) | `192.168.X.___` | Sätts vid Proxmox-installation |
| CT 101 — cloudflared | `192.168.X.___` | |
| CT 102 — NPM | `192.168.X.___` | |
| CT 103 — Frigate | `192.168.X.___` | |
| VM 100 — Home Assistant | `192.168.X.___` | Tilldelas via DHCP, reserveras i routern |

> **Tips:** Om din router stöder det, skapa DHCP-reservationerna redan nu (med "dummy" MAC-adresser som du uppdaterar senare). Annars, skriv bara ner dina valda adresser.

## 7. Förbered kamerorna

**Vad:** Om dina Axis-kameror redan är installerade och anslutna till nätverket kan du förbereda dem nu.

**Gör nu:**
1. **Bestäm ett gemensamt lösenord** för alla kameror (eller ett per kamera om du föredrar det). Skriv ner det.
2. **Ge varje kamera en statisk IP** (antingen via DHCP-reservation i routern eller direkt i kamerans webbgränssnitt).
3. **Notera modell och IP** för varje kamera:

| Kamera | Modell | IP-adress | Plats |
|--------|--------|-----------|-------|
| Kamera 1 | | `192.168.X.___` | |
| Kamera 2 | | `192.168.X.___` | |
| ... | | | |

4. **Skapa stream-profiler** (valfritt, kan göras senare): Om du vill kan du redan nu logga in på varje kamera och skapa profilerna `main` och `detect` enligt [guide 07](07-axis-kameror.md). Alternativt kan du köra automatiseringsskriptet senare.

## 8. Fyll i TOKENS.md

**Vad:** Alla känsliga nycklar och lösenord samlas på ett ställe så du slipper leta mitt i installationen.

**Gör nu:**
1. Kopiera `TOKENS.md.example` till `TOKENS.md` (denna fil ignoreras av git och pushas aldrig).
2. Fyll i allt du har samlat:

```
# Mina tokens och nycklar (PUSHAS ALDRIG TILL GIT)

## Cloudflare
- Konto-email: ___
- Domän: ___
- Tunnel-token: ey___
- Tunnel-namn: ___

## Subdomäner
- Home Assistant: ha.___
- Frigate: frigate.___
- NPM: npm.___

## Proxmox
- IP: 192.168.X.___
- Root-lösenord: ___ (välj ett starkt lösenord)

## Kameror
- Användarnamn: ___
- Lösenord: ___

## Home Assistant
- MQTT-användarnamn: mqtt-user
- MQTT-lösenord: ___

## Frigate
- Admin-lösenord: ___ (genereras vid första start, fyll i då)
- RTSP-lösenord (kameror): ___
```

## Checklista — Redo för installationsdag?

Bocka av allt du gjort:

- [ ] Domän aktiv i Cloudflare (status: "Active")
- [ ] Cloudflare Tunnel skapad (token sparad)
- [ ] Cloudflare Access-policyer skapade för frigate + npm
- [ ] Proxmox ISO nedladdad och flashad till USB
- [ ] RAM beställd/levererad (16 GB DDR4 SODIMM)
- [ ] Lagringsdisk beställd/levererad (1 TB SSD)
- [ ] IP-plan bestämd och nedskriven
- [ ] Kameror har statiska IP:er och kända lösenord
- [ ] TOKENS.md ifylld med allt ovan

**När allt ovan är klart och din OptiPlex har kommit, gå vidare till [Steg 01: BIOS-konfiguration](01-bios-setup.md).**
